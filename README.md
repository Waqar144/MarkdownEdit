# MarkdownEdit 
MarkdownEdit, as the name suggests, is a simple and easy program to create and edit Markdown files.

# Features:
- Save and load Markdown files
- Undo-Redo
- Preview Markdown files
- Auto complete(Close brackets, list, ...)
- Find/Replace
- Export to HTML
- Print
